*Generated from Problem Statement: 2026-06-16*

## Summary

A Christmas tree ornament with an embedded MFRC522 RFID reader and two NeoPixel rings. Tapping a Disney MagicBand (or any RFID card) triggers a light animation and toggles a relay in a floor-level power box that switches 120V mains to the tree lights. The ornament and power box are connected by a discrete hidden wire.

## Scope

### In:
- Arduino firmware (RFID detection, NeoPixel animations, relay control)
- Code cleanup and bug fixes on existing sketch
- Optional: UID whitelist to restrict access to specific bands
- Optional: DFPlayer Mini audio cue (currently disabled due to pin conflict)
- Hardware documentation (pin map, wiring, BOM)

### Out:
- Enclosure design / 3D printing (separate effort)
- PCB design
- Network/wireless integration
